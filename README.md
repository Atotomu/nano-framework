Java MVC + ORM框架 Nano Framework
====

	Nano Framework基于Google Guice框架进行开发，使用Guice的IoC和AOP特性可以快速的定义和开发组件及服务。
	设计初衷是减少代码量，让开发人员专注于业务层代码的开发。
